monnix
======

## コレは何？
Zabbixの障害数をシンプルに表示するツールです
 * zabbix server 2.0.9のAPIを利用しています
 * 開発運用の現場に設置した大型モニタに表示しておくために作りました
 * 時間を見て改良していきますが動作は無保証でお願い致します。
